gmt.show¶

gmt.show(dpi=100, width=500, return_img=False)[source]¶

Display the last figure in the Jupyter notebook

You will need to have IPython installed for this to work. You should have it if you are using a Jupyter notebook.

Parameters:

dpi : int

The image resolution (dots per inch).

width : int

Width of the figure shown in the notebook in pixels.

return_img : bool

Whether or not to return the IPython.display.Image variable.
