gmt.figure¶

gmt.figure()[source]¶

Start a new figure.

All plotting commands run afterward will append to this figure.

Unlike the command-line version (gmt figure), this function does not trigger the generation of a figure file. An explicit call to gmt.savefig or gmt.psconvert must be made in order to get a file.
