API Reference¶

High-level functions for GMT modules¶

Each GMT module (gmt pscoast, gmt psbasemap, etc.) is wrapped by a function in the gmt top-level module.

gmt.figure() Start a new figure.
gmt.show([dpi, width, return_img]) Display the last figure in the Jupyter notebook
gmt.psbasemap(**kwargs) Produce a basemap for the figure.
gmt.pscoast(**kwargs) Plot continents, shorelines, rivers, and borders on maps
gmt.psconvert(**kwargs) Convert [E]PS file(s) to other formats.
gmt.psxy(data, **kwargs) Plot lines, polygons, and symbols on maps.

Additional utility functions:

gmt.test([doctest, verbose, coverage, figures]) Run the test suite.

Low-level wrappers for the GMT C API¶

The GMT C API is accessed using ctypes. The gmt.clib module offers functions and classes that wrap the C API with a pythonic interface.

Functions¶

gmt.clib.call_module(session, module, args) Call a GMT module with the given arguments.
gmt.clib.create_session() Create the GMTAPI_CTRL struct required by the GMT C API functions.
gmt.clib.destroy_session(session) Terminate and free the memory of a registered GMTAPI_CTRL session.
gmt.clib.load_libgmt() Find and load libgmt as a ctypes.CDLL.

Classes¶

gmt.clib.APISession() Context manager to create a GMT C API session and destroy it.
